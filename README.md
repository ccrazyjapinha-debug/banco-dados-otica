# Sistema Ótica -

Projeto de Banco de Dados para gerenciamento de uma ótica (Clientes, Receitas, Vendas e Estoque).

## Funcionalidades
* **Estrutura:** Criação completa das tabelas e relacionamentos.
* **Dados:** Scripts de inserção (INSERT), atualização (UPDATE) e exclusão (DELETE) controlada.
* **Consultas:** Relatórios de vendas e listagem de clientes.

##  Como rodar
1.  Baixe o arquivo `script_otica_atividade4.sql` deste repositório.
2.  Abra no **MySQL Workbench**.
3.  Execute o script inteiro.
